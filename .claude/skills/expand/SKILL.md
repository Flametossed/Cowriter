# expand

1. Pull the target beat/section from `outline.md`.
2. Resolve the context stack per [context-map.md](../../../frameworks/core/context-map.md) — fiction: bible + fingerprint/style-guide + slop-markers, as before; non-fiction: brief + facts + voice + slop floor.
3. Follow the **expand** section of [drafting-methods.md](../../../frameworks/drafting/drafting-methods.md) (fiction), or Phase 5 of [nonfiction-method.md](../../../frameworks/nonfiction/nonfiction-method.md) (claim/evidence/so-what/handoff).
4. Keep the beat's turn (or section's claim) intact; flag thin beats/sections rather than padding.
5. Write to `drafts/` with the new-facts / new-claims block.
