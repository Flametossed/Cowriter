# dialogue

1. Load each speaker's voice profile from `bible/characters.md`.
2. Follow the **dialogue** section of [drafting-methods.md](../../../frameworks/drafting/drafting-methods.md).
3. Test: tags stripped, can you still tell who's speaking? If not, voices aren't distinct.
4. Subtext carries weight; no line explaining what the previous line meant.
5. Generate the exchange, or for a sharpen pass flag + rewrite only the weak lines.
