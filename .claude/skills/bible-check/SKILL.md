# bible-check

Query the story bible.

1. Find the book's `bible/`.
2. Follow the **bible-check** section of [bible-method.md](../../../frameworks/bible/bible-method.md).
3. Lookup order: `canon-facts.md` → topical file (`characters.md`/`world.md`) → `timeline.md`.

Answer with a citation, or say plainly it's underspecified and offer to establish + record it via `/bible-update`. Do not guess.
