# continuity

Check a draft against canon.

1. Load the draft + the book's `bible/` (`canon-facts.md`, `characters.md`, `world.md`, `timeline.md`).
2. Run every check in [continuity-method.md](../../../frameworks/revision/continuity-method.md): locked facts, spellings, chronology, knowledge ledger, voice, world rules, open threads.
3. Output the CONTINUITY REPORT: **Bugs / Risks / Voice notes / Threads.**

Report, don't fix silently. If the draft is right and the bible is stale, route the fix through `/bible-update` (possibly a retcon).
