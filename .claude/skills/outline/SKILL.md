# outline

Generate the plot outline.

1. Load `bible/premise.md` and `bible/characters.md` for the book.
2. Confirm structure with the user (default **3-act** if unset).
3. Follow [outline-method.md](../../../frameworks/prewriting/outline-method.md): generate beats, then pressure-test escalation, arc movement, and setup/payoff against the bible.
4. Write `projects/<book>/outline.md`.

Flag the weakest beat and offer alternatives. Outline is revisable, not a contract.
