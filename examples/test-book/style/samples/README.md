# Samples

Drop writing samples here as `.md` or `.txt` — one passage per file is cleanest.

Two kinds:
- **Your own prose** → name it `mine-*.md`. Mode: `match-mine`. The AI extends *your* voice.
- **An author you want to write like** → name it `comp-<author>-*.md`. Mode: `emulate-comp`. The AI targets the *technique*, never copies the text.

After adding/changing samples, run `/style-learn <book>` to regenerate `../style-fingerprint.md`.

More samples = sharper fingerprint. Aim for 3–5 passages, 300+ words each.
