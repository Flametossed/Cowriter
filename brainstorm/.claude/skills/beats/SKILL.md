# beats

Subvert the turns. Follow [beat-method.md](../../../frameworks/ideation/beat-method.md).

1. List the major turns the plot **engine** produces (not a structure's beat names) — usually 5–8.
2. For each turn, run the three-move drill: **name the default move → generate 2 alternatives (same job, different move) → pick the one with heat the engine justifies.**
3. Run the **predictability check** — could a reader predict this beat from the previous one? Aim for surprising-yet-earned.
4. **Protect the payoffs** — every surprising beat must still pay off an earlier setup. Trope-check the sequence (resurrection-undo, villain monologue, dream-reveal).
5. Log the beat sketch (job / default move / chosen move / pays off). It hands to Cowriter's `outline-method`, which chooses structure and expands the beats.
