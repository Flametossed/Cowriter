# plot

Build the engine. Follow [plot-method.md](../../../frameworks/ideation/plot-method.md).

1. Read the locked premise (logline + three-layer conflict + central cost) from the session log.
2. Find the **irreversible first move** — what early act closes the safe exit.
3. Generate **3 candidate engines**, each with a different escalation logic (world-cost / loyalty-squeeze / protagonist-flaw). State each as *"Every time X, it forces Y, until Z is unavoidable."*
4. **Pick by the flaw test** — the engine where plot pressure *is* the internal Lie under stress (the seam Cowriter's character method picks up).
5. **Trope-check the spine** (mentor death? undone sacrifice? final duel?) and pressure-test escalation five steps. Log the engine + first move + squeeze.
